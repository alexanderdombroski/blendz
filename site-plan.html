Overview

Purpose

This is a site for a smoothie place called Blendz. There is a menu with prices, and other pages. This site allows users to view the menu or order for pickup or delivery.

Audience

We have smoothies for anyone, whether they have allergies, are seeking healthy solutions, or have a really big sweet tooth.

Dynamic elements

Menus will pop down with more information. JavaScript allows a form to be submitted. and if I have time emails can be sent

Wireframes

Home

Text to the side of page buttons will have animation. For example, the text near "menu" will change to see today's special or experiment with combinations. The image in the footer will be a popup that dispays different share icons. In desktop view, these icons will be spread across the footer.

Menu

When a menu item image is clicked it will expand to show more info and a button to add it to cart. The is split in two, build your own or order a prebuilt. The layouts of both will be very similar.
