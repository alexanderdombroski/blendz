Overview

Purpose

This is a site for a smoothie place called Blendz. There will be a menu with prices, and other pages that help customers have a better experience with the site. This site allows users to view the menu, see prices, or order for pickup.

Audience

We have smoothies for anyone, whether they have allergies, are seeking healthy solutions, or want to fully customize their smoothie.

Dynamic elements

The Menu will load in all menu items, and menu items can pop open more information when clicked on. Home page will have some dynamic text that will cycle through phrases when hovered over. The footer's social media icons will pop up and down when clicked.

Wireframes

Home

Text to the side of page buttons will have animation. For example, the text near "menu" will change to see today's special or experiment with combinations. The image in the footer will be a popup that dispays different share icons. In desktop view, these icons will be spread across the footer.

Menu

When a menu item image is clicked it will expand to show more info and a button to add it to cart. The is split in two, build your own or order a prebuilt. The layouts of both will be very similar.
